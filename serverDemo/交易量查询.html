交易量查询相关需求规则：

需求点

规则文案

备注

查询

查询条件：输入框（客户姓名、手机号、客户账号、订单号、信号源账号）、下拉框_信号源名称、下拉框_交易类型、时间控件

规则：

单查询条件：

输入框【客户姓名、手机号、客户账号、订单号、信号源账号】

下拉框_信号源名称

下拉框_交易类型

时间控件

多条件组合查询：

两条件组合：

输入框 + 下拉框_信号源名称

输入框 + 下拉框_交易类型

输入框 + 时间控件

下拉框_信号源名称 + 下拉框_交易类型

下拉框_信号源名称 + 时间控件

下拉框_交易类型 + 时间控件

三条件组合：

输入框 + 下拉框_信号源名称 + 下拉框_交易类型

输入框 + 下拉框_信号源名称 + 时间控件

输入框 + 下拉框_交易类型 + 时间控件

下拉框_信号源名称 + 下拉框_交易类型 + 时间控件

四条件组合：

输入框 + 下拉框_信号源名称 + 下拉框_交易类型 + 时间控件

1、查询条件暂定为：输入框【客户姓名、手机号、客户账号、订单号、信号源账号】、下拉框_信号源名称、下拉框_交易类型、时间控件，四者可分别进行单条件查询、两两组合查询、三条件组合查询、四条件同时查询操作；

2、输入框中可查询的字段有：客户姓名、手机号、客户账号、订单号、信号源账号；

3、组合查询时，若有匹配的数据，则显示在查询结果列表中，若无，则显示无对应结果；

4、复制内容粘贴输入框进行查询时，需判断前后置空格；

5、未设置任何查询条件，直接查询，列表显示当前系统内所有交易记录

输入框

可查询字段：

客户姓名、手机号、客户账号、订单号、信号源账号

规则：

1、输入框单独查询：

1.1、输入框中只能输入单个字段，不能同时输入多个字段进行查询；

1.2、输入客户姓名“张三”，点击【查询】，查询结果显示客户姓名为张三的所有交易记录；

1.3、输入手机号“13812345678”，点击【查询】，查询结果显示手机号为“13812345678”的客户名下所有账号的交易记录；

1.4、输入客户账号“4408330623”，点击【查询】，查询结果显示客户账号为4408330623的所有交易记录；

1.5、输入订单号“#92145874”，点击【查询】，查询结果显示对应的订单数据；

1.6、输入信号源账号“2000133027”，点击【查询】，查询结果显示跟随该信号源的所有交易记录；

1.7、输入框留空，且不选择其他查询条件直接查询，查询结果显示跟单系统内所有的交易记录；

2、输入框结合其他查询条件组合查询：

组合查询时，若有匹配的数据，则显示在查询结果列表中，若无，则显示无对应结果

未设置任何查询条件，直接查询，列表显示当前系统内所有交易记录

下拉框_信号源名称

预设字段：

信号源名称

规则：

预先将系统内所有信号源名称添加到下拉数据源中（信号源一、信号源二...），此下拉选项结合输入框，可查询客户在其跟随的信号源下面所生成的所有交易记录

1、在查询输入框输入客户姓名或者账号，在信号源名称下拉框中选择客户跟随的信号源，再点击【查询】，则显示该客户在跟随该信号源时生成的所有交易记录；

2、输入框留空，只选择信号源点击【查询】，则显示跟随该信号源的所有客户在跟随期间生成的所有交易记录

下拉框_交易类型

预设字段：

交易类型

规则：

1、输入框输入客户姓名xxx：

选择buy点击【查询】，即对客户xxx跟随的所有信号源产生的所有buy单进行查询统计；选择sell点击【查询】，即对客户xxx跟随的所有信号源产生的所有sell单进行查询统计；

2、输入框输入客户姓名xxx，信号源名称下拉框中选择客户跟随的信号源“信号源一”：

选择buy点击【查询】，即对客户xxx跟随信号源一时产生的所有buy单进行查询统计；选择sell点击【查询】，即对客户xxx跟随信号源一时产生的所有sell单进行查询统计；

3、输入框留空：

选择buy点击【查询】，对跟单系统内所有buy单进行查询统计；

选择sell点击【查询】，对跟单系统内所有sell单进行查询统计


1、输入框留空时的查询，即是针对系统内所有客户、信号源的交易类型的查询；

2、输入框不留空，则是指定客户或者指定信号源的条件下对交易类型的查询

时间控件

1、开始时间<结束时间；

2、时间控件限定查询的时段，若不选择开始时间/结束时间，则查询系统内所有时间段内的交易记录；

3、输入框输入客户姓名xxx，信号源名称下拉框中选择客户跟随的信号源“信号源一”，交易类型下拉框中选择buy，再选择一个时段（开始时间<结束时间）点击【查询】，则查询系统在选择的时间段内，客户xxx跟随信号源一时产生的所有buy单记录

时间控件的处理：开始时间<结束时间，若输入异常，则限制或给出提示信息

导出excel

入口：

交易量列表上方【导出excel】按钮

规则：

导出客户跟随信号源产生的交易记录数据，方便日后做数据统计使用

导出的excel文件不能出现乱码，导出数据须符合需求
